## Topic

Migration und öffentliche Meinung in Europa Untersuchung des Zusammenhangs zwischen der tatsächlichen Zuwanderungsrate und den Einstellungen der Bevölkerung gegenüber Migration in den EU-Mitgliedsstaaten.

## Data Selection

[EuroStat Immigration](https://ec.europa.eu/eurostat/databrowser/view/migr_imm1ctz/default/table?lang=en&category=migr.migr_cit.migr_immi)

[Migration Data Portal](https://www.migrationdataportal.org/themes/public-opinion-migration)

Idea - Twitter API 

## Research Question

H1: Higher immigration rates correlate with more negative public attitudes toward migration.

H2: The relationship weakens or reverses in countries with a longer history of immigration.
